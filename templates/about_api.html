Эндпоинт API

GET
/api/user/<id>

Возвращает информацию о пользователе по указанному ID. Если пользователь с таким ID не найден, возвращается null.

Параметры запроса
Параметр Тип Обязательный Описание
id Integer Да Уникальный идентификатор пользователя
Пример запроса
# Получение пользователя с ID=42
GET /api/user/42
Host: example.com
Accept: application/json
Пример ответа
{
  "id": 42,
  "name": "Иван Петров",
  "about": "Разработчик ПО с 5-летним опытом",
  "email": "ivan@example.com",
  "hashed_password": "password",
  "created_date": "2023-10-15T14:32:10"
}

Модель данных User

Структура таблицы "users"

Таблица содержит информацию о зарегистрированных пользователях системы:

Поле Тип данных Описание Ограничения
id Integer Уникальный идентификатор Primary Key, Auto Increment
name String Имя пользователя Unique, Nullable
about String Информация о пользователе Nullable
email String Адрес электронной почты Index, Unique, Nullable
hashed_password String Хэшированный пароль Nullable
created_date DateTime Дата регистрации Default: текущая дата
